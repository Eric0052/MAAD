The functional requirements are:

1. **[FR-001] User Interaction and Learning:**
   1.1 The system will present a series of fraction questions in a storyline format.
   1.2 The system will provide feedback based on students' scores at the end of the session.
   1.3 The system will offer a dynamic and adaptive storyline based on user progress and performance.
   1.4 The system will engage players with multiple-choice questions integrated within a storyline, providing feedback and reinforcement.

2. **[FR-002] System Accessibility and Interface:**
   2.1 The system will be accessible via any Internet-connected computer with a modern web browser.
   2.2 The system will feature an introductory movie to set up the storyline, which can be skipped by the user.
   2.3 The main menu will include a help section, a link to the Denominators' web page, and a brief team summary.
   2.4 The main menu will serve as the initial interface, offering options to start the game or access external educational resources.

3. **[FR-003] Administrative Capabilities:**
   3.1 The system will allow an administrator to update fraction questions via a web-accessible Question Updater tool.
   3.2 The Question Updater will require a password for access and will feature a user-friendly interface for modifying questions.

4. **[FR-004] Input and Feedback Mechanisms:**
   4.1 The system will process player inputs during gameplay for calculating spaceship velocity adjustments based on fraction inputs.
   4.2 The system will validate fraction inputs to ensure they are integers and that the denominator is not zero.
   4.3 The system will provide immediate feedback for invalid inputs, requesting a new input from the player.

5. **[FR-005] System Security and Maintenance:**
   5.1 The system will be secure, relying on modern web security standards.
   5.2 Maintainability will be a primary goal, with the system designed for easy updates and modifications.

6. **[FR-006] External Resources and Support:**
   6.1 The Math Umbrella component will provide links to external educational resources, opening in a separate window.

7. **[FR-007] System Conclusion and Replay Options:**
   7.1 The system will conclude with an ending scene where the user's score is calculated and ranked, offering an option to quit or try again.
